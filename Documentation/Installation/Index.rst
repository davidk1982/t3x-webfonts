============
Installation
============


Composer installation (recommended)
-----------------------------------

If your TYPO3 instance is based on modern composer installation, just require the package via composer:

.. code-block:: bash

   composer req zotornit/webfonts


Extension Manager installation
------------------------------

Switch to the backend module **Extensions**. Press the **Retrieve/Update** button and
search for the extension key :code:`webfonts` and import the extension from the repository.


Manual download from TER
------------------------------

**Get it from typo3.org:** You can always get current version from
`http://typo3.org/extensions/repository/view/webfonts/current/
<http://typo3.org/extensions/repository/view/webfonts/current/>`_ by
downloading either the t3x or zip version. Upload
the file afterwards in the **Extension Manager**.


Latest version from git
-----------------------

.. code-block:: bash

   git clone https://github.com/zotornit/t3x-webfonts
