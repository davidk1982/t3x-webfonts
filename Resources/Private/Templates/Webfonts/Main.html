Just add the following code to your typoscript. The extension will do the rest.

plugin.tx_webfonts.settings {
  fonts {
    20 {
      id=fontawesome
      provider=fontawesome
      version=5.13.0
      methods=css
      variants=all
      minified=true
    }
  }
}

Visit extension documentation for a detailed parameter description
